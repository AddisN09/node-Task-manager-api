 Task Manager API (Pure Node.js)

A backend Task Management API built with vanilla Node.js to understand core backend architecture without frameworks.

📌 About The Project

This project is a pure Node.js Task Manager API built from scratch (no Express or external frameworks).

It was created as a learning exercise to deeply understand:

How backend systems work internally
How routing is handled manually
Session-based authentication
Role-based system design (Admin & Users)
✨ Features
🔐 Authentication
Session-based authentication
Login & logout system
Role-based access control
👨‍💼 Admin
Create and manage users
Create, assign, and manage tasks
Approve or reject task completion requests
👤 Users
View assigned tasks
Request approval for completed tasks
Update and manage their profile
✅ Task Workflow
Admin creates and assigns tasks
User works on assigned task
User sends completion request
Admin reviews and approves/rejects
🛠️ Built With
Node.js (Vanilla)
JavaScript
Custom session handling
